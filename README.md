# Netflix Data Analysis using Pandas

## Project Overview
This project analyzes Netflix movies and TV shows dataset using Python Pandas.

## Skills Used
- Pandas
- Data Cleaning
- Filtering
- Aggregation
- Visualization
- Exploratory Data Analysis (EDA)

## Questions Analyzed
- Movies vs TV Shows
- Most common ratings
- Top countries producing content
- Release year trends
- Missing value analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.
